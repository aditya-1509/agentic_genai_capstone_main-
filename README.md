# Intelligent Credit Risk Scoring & Agentic Lending System

A production-level AI lending decision support assistant built with LangGraph, Groq LLM, RAG, and multi-layer guardrails.

## Architecture

```
User Input → ML Model → LangGraph Agent → RAG → LLM Reasoning → Guardrails → UI
```

### Components

| Layer | Technology | Purpose |
|---|---|---|
| ML Prediction | Scikit-learn / XGBoost | Computes default probability from borrower data |
| Agent Orchestration | LangGraph | 8-node stateful workflow |
| RAG | ChromaDB + sentence-transformers | Fetches relevant RBI and credit guidelines |
| LLM Reasoning | Groq (llama-3.3-70b-versatile) | Generates personalized narrative |
| Domain Guard | Groq (llama3-8b-8192) | Classifies if input is credit-domain |
| UI | Streamlit | Interactive dashboard with what-if analysis |

### Guardrail Layers

1. **Input Guardrail** — Validates required credit profile fields
2. **Domain Guardrail** — LLM classifier blocks off-topic requests
3. **Reasoning Guardrail** — System prompt + output keyword scan
4. **Hybrid Decision Guardrail** — Strict rule overrides LLM: `>0.7=Reject`, `0.4-0.7=Review`, `<0.4=Approve`
5. **Output Guardrail** — Enforces structured JSON output format

## Running Locally

```bash
# 1. Clone and install
git clone <your-repo>
cd gen_ai_capstone-main
pip install -r requirements.txt

# 2. Set your Groq API key
echo "GROQ_API_KEY=your_key_here" > .env

# 3. Build the RAG vector database (runs automatically on first app start too)
python rag/build_index.py

# 4. Launch the app
streamlit run app.py
```

## Deploying to Streamlit Cloud

1. Push the repository to GitHub (including the `rag/chroma_db/` folder).
2. Go to [share.streamlit.io](https://share.streamlit.io) and connect your repo.
3. Set `app_file = app.py`.
4. In **Secrets**, add:
   ```toml
   GROQ_API_KEY = "your_groq_api_key"
   ```
5. Deploy. The RAG index is pre-built, so startup is fast.

## Deploying to Hugging Face Spaces

The YAML header at the top of this README configures the Space automatically.

1. Create a new Space at [huggingface.co/spaces](https://huggingface.co/spaces).
2. Set SDK to **Streamlit**.
3. Push the full repository:
   ```bash
   git remote add space https://huggingface.co/spaces/<your-username>/<space-name>
   git push space main
   ```
4. In Space **Settings > Secrets**, add `GROQ_API_KEY`.

## Key Features

- **LangGraph agentic workflow** with explicit state management across all nodes
- **Real feature importance** from trained model drives risk driver explanation (not hard-coded rules)
- **Personalized Story Mode** narrative generated by llama-3.3-70b-versatile
- **Live What-If Analysis** — instant ML recalculation without LLM (slider-driven)
- **RAG with MMR retrieval** for diverse, non-redundant regulatory citations
- **4-layer domain scope guardrail** prevents any off-topic LLM responses
- **Downloadable structured report** for audit trail
- **Dual LLM strategy** — fast 8b classifier + powerful 70b reasoner
